Sobre o Fabulio

Saiba mais sobre nossa plataforma e os créditos das APIs utilizadas.

O Fabulio é uma plataforma de streaming que reúne informações sobre filmes e séries, permitindo que você explore, descubra e organize seus conteúdos favoritos em um só lugar.

Funcionalidades

  • Catálogo: Explore filmes e séries com avaliações, sinopses e detalhes completos.
  • Gêneros: Navegue por categorias como Ação, Comédia, Drama, Terror e muito mais.
  • Lista de Favoritos: Salve seus filmes e séries preferidos em uma lista personalizada.
  • Busca: Encontre rapidamente o que você procura.
  • Detalhes: Informações completas sobre cada título, incluindo elenco, diretor e avaliação.

Tecnologias Utilizadas

  • Front-end: HTML5, CSS3, JavaScript
  • Back-end: Java com Spring Boot
  • Banco de Dados: PostgreSQL
  • API Externas: OMDB, Fanart, TMDB

Créditos da API

Este produto utiliza a API do The Movie Database (TMDB).

utilizada para converter imdb_id em tvdb_id, e para obter as imagens das séries via FanArt api.
